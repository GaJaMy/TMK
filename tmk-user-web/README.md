# tmk-user-web

사용자용 웹 애플리케이션 디렉터리입니다.

## 목적

- 회원가입, 로그인, 토큰 재발급
- 내 문서 등록 및 문제 생성
- 시험 생성, 시작, 응시, 결과 확인

## 현재 상태

- 정적 프로토타입 기준 랜딩페이지와 로그인 화면 작성
- 프론트 기술 스택은 아직 미확정
- API 연동 전
- 요구사항은 `docs/사용자 웹 요구사항.md`에 정리

## 비고

- 이 디렉터리는 Gradle 모듈이 아니라 별도 웹 앱 디렉터리로 운영합니다.
- 기존 `tmk-api/src/main/resources/static` 정적 페이지는 점진적으로 대체될 예정입니다.
