# tmk-admin-web

관리자용 웹 애플리케이션 디렉터리입니다.

## 현재 포함 화면

- 관리자 로그인 페이지: `index.html`
- 관리자 메인페이지: `dashboard.html`
- 관리자 관리 페이지: `admin-create.html`
- 공용문제 관리 페이지: `questions-list.html`

## 목적

- 관리자 인증 진입점 제공
- 관리자 메인 모니터링 화면 제공
- 공용문제 관리 화면 제공
- 관리자 관리 화면 제공

## 비고

- 현재는 정적 프로토타입 기준이다.
- API 연동 전 단계이므로 로그인 제출은 프론트 상호작용만 제공한다.
